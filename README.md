# Goalsetter MERN app

This is the goalsetter app from the [MERN Stack Web App](https://github.com/Babanila/mern-stack).

## Usage

Rename the .env-example to .env and add your NODE_ENV, PORT and MONGO_URI

### Install dependencies

```
# Backend deps
npm install


# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```
